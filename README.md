# 🎮 Hangman Game (Python)

## 📌 Overview

This is a simple **text-based Hangman Game** built using Python as part of my **CodeAlpha Python Internship (Task 1)**.

The game randomly selects a word, and the player must guess it letter by letter within a limited number of attempts.

---

## 🚀 Features

* 🎯 Random word selection from predefined list
* 🔤 Letter-by-letter guessing system
* ❌ Limited wrong attempts (6 chances)
* 🧍 Visual Hangman stages (ASCII art)
* 📊 Displays progress after each guess
* 🏆 Win/Lose result at the end

---

## 🛠️ Tech Stack

* **Language:** Python
* **Concepts Used:**

  * Loops
  * Conditional Statements
  * Strings & Sets
  * Random Module
  * Input Validation

## 🎮 How to Play

* The game will choose a random word
* Enter one letter at a time
* Correct guesses reveal the word
* Wrong guesses reduce your chances
* You have **6 wrong attempts**
* Guess the word before attempts run out!
